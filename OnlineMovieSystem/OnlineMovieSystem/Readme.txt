-The "User" class represents basic user properties. The "RegularUser" and "SubscriptionUser" classes are derived from this base class. "SubscribedUser" represents subscribed users and includes behavior specific to subscribed users.

-The "movie" class represents movies by attributes such as ID, title, and price.

-Both the "Regular User" and "Subscribed User" classes contain methods for renting or purchasing movies.

-The "RequestedMovie" class represents requested movies that are not currently available in the system. These movies can be added to the system if desired. This class is not bound to a particular user type.
