Class diagram:

-The "AirlineCompany" class has an added attribute "name" to hold the names of airline companies.

-The "departureAirport" and "arrivalAirport" attributes of the "Flight" class are of type "Airport", indicating that they point to instances of the "Airport" class, representing the departure and arrival airports of a flight, respectively.

-The "Airplane" class has a new attribute "pilots" to denote that an airplane can be associated with one or more pilots.

-The "Pilot" class includes a new method "getAssignedFlights()" which retrieves the flights assigned to a pilot.
