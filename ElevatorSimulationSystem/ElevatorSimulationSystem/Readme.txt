-The "Building" class represents the building and can contain multiple elevators.

-The "Elevator" class represents elevators, including attributes such as capacity, status, and current floor information. It also includes behaviors like movement, door opening/closing.

-The "Floor" class represents floors, with attributes like floor number and call buttons.

-The "CallButton" class represents call buttons, including direction information.

-The "ArrivalBell" class represents arrival bells, containing information about the light status.
